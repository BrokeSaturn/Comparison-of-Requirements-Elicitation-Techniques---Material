## 📌 Sobre el Proyecto

Este repositorio consolida la investigación y el levantamiento bibliográfico estructurado que hemos realizado como equipo. Nuestro objetivo principal es documentar las metodologías más recientes y relevantes en el campo de la **Ingeniería de Requisitos**, con especial énfasis en las técnicas de elicitación, análisis en entornos ágiles e integración de nuevas tecnologías como IoT.

## 📂 Estructura de las Librerías

La bibliografía ha sido debidamente curada, exportada en formato `.bib` y clasificada según la base de datos científica de origen para facilitar su integración con gestores de referencias (como Mendeley o Zotero).

| Archivo | Descripción |
| :--- | :--- |
| `Scopus_Final.bib` | Librería con fuentes de alto impacto extraídas de Scopus. |
| `Sience_Direct_Final.bib` | Material de investigación obtenido de Science Direct. |
| `Springer_Final.bib` | Artículos y journals académicos extraídos de Springer. |
